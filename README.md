# BEGAN-pytorch
pytorch implementation of BEGAN

## Packages needed:
`pip install tensorboard-pytorch`


## Training data preparation: 
https://github.com/Heumi/BEGAN-tensorflow  #Make Train Data

## Run:
start training:

`python began.py`

after training:

`tensorboard --logdir runs`
